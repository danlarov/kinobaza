## Сообщить об уязвимости

Если вы обнаружили уязвимость безопасности в КиноБаза:

1. **Не создавайте публичный Issue** — это может навредить пользователям
2. Опишите уязвимость в личном сообщении через Gitverse
3. Укажи:
   - Описание проблемы
   - Шаги для воспроизведения
   - Возможные последствия

Мы постараемся ответить в течение 72 часов и исправить проблему в ближайшем обновлении.

## Важные замечания

- Не храните API ключи в коде — используйте `.env` файл
- Файл `.env` добавлен в `.gitignore` и не попадает в репозиторий
- Для работы приложения нужен собственный API ключ
