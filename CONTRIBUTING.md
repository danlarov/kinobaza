# Правила участия

Спасибо за интерес к проекту КиноБаза! 🎬

## Как внести вклад

### Сообщить об ошибке
1. Проверь что такой Issue ещё не создан
2. Создай новый Issue с описанием ошибки
3. Укажи шаги для воспроизведения
4. Приложи скриншот если возможно

### Предложить улучшение
1. Создай Issue с тегом `enhancement`
2. Опиши что хочешь улучшить и зачем
3. Дождись обратной связи

### Отправить код (Pull Request)
1. Сделай Fork репозитория
2. Создай ветку: `git checkout -b feature/название`
3. Внеси изменения
4. Сделай коммит: `git commit -m "Описание изменений"`
5. Отправь: `git push origin feature/название`
6. Создай Pull Request

## Требования к коду
- Следуй структуре Clean Architecture проекта
- Используй BLoC для управления состоянием
- Пиши понятные названия переменных и методов
- Проверь что приложение запускается без ошибок
